# primevue_array_validate

使用  Primevue 的 datatable 配合 Vee-validate 驗證 多陣列物件。


## 專案初始

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
